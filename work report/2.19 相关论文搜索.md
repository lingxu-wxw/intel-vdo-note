#### 2.19 相关论文搜索

---

```
找论文的方法：一般找论文有两个方法
1. 一个是在相关会议上找，比如存储相关FAST，VLDB，或者数据相关的ICDE，或者系统综合的ATC这些，另一个就是google scholar上搜一些关键词，比如block layer compression，或者hardware compression等等。
2. 还有一种就是在读过的文章的参考文献中找论文
```

```
老李的说法：
1. 组内小康ATC的论文：QZFS
2. FAST今年新发的一两篇文章，有关压缩
3. 郭鹤林的文章，在组会上讲过的文章
4. 压缩分的层面：block层，file system层，application层，每个大概就个位数篇，relate work，discussion
5. 学校发布的学生无需VPN远程连接数据库的方法：官微发过
6. FAST的官网：官网上有会议的很多材料，比如开会的时候演讲的ppt或者视频，论文的final version
7. 找reference
8. 还有一个网站可以搜计算机相关的论文
```
